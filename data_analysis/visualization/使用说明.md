# C题常用可视化脚本使用说明

## 📁 文件列表

| 文件名 | 功能 | 适用场景 |
|-------|------|---------|
| `折线图_时间序列.py` | 展示数据随时间变化趋势 | 用户行为趋势、销售预测、时间序列分析 |
| `柱状图_分组对比.py` | 对比多个模型/方法的性能 | 算法对比、模型评估、方案比较 |

| `热力图_相关矩阵.py` | 展示多特征间的相关性 | 特征选择、降维分析、变量筛选 |
| `平行坐标图_高维数据.py` | 高维数据可视化、客户分群 | 用户分类、多目标优化、聚类分析 |

在脚本中找到这一段：

## 💡 常见问题

### Q1: 中文显示乱码怎么办？

**Mac系统**：
```python
plt.rcParams['font.sans-serif'] = ['Arial Unicode MS']
```

**Windows系统**：
```python
plt.rcParams['font.sans-serif'] = ['SimHei']
```

### Q2: 如何修改图表大小？

找到这一行：
```python
fig, ax = plt.subplots(figsize=(12, 6), dpi=300)
#                               ↑宽  ↑高     ↑分辨率
```

修改 `figsize` 参数即可。

### Q3: 如何修改颜色？

**方法1**: 使用颜色名称
```python
color='red'  # 或 'blue', 'green', 'orange' 等
```

**方法2**: 使用十六进制颜色代码
```python
color='#2E86AB'  # 推荐使用，颜色更专业
```

**推荐配色**：
- `#2E86AB` - 专业蓝
- `#F18F01` - 活力橙
- `#6A994E` - 自然绿
- `#A23B72` - 典雅紫
- `#C73E1D` - 警示红

### Q4: 如何保存为其他格式？

修改保存文件名的后缀：
```python
plt.savefig('我的图表.png', dpi=300, bbox_inches='tight')  # PNG格式
plt.savefig('我的图表.pdf', dpi=300, bbox_inches='tight')  # PDF格式（推荐论文使用）
plt.savefig('我的图表.svg', bbox_inches='tight')  # SVG格式（可编辑）
```

### Q5: 图例位置如何调整？

```python
ax.legend(loc='upper right')  # 位置选项
```

**可选位置**：
- `'upper right'` - 右上角
- `'upper left'` - 左上角
- `'lower right'` - 右下角
- `'lower left'` - 左下角
- `'best'` - 自动选择最佳位置

## 🎨 美赛O奖制图要点

### ✅ 必须做到的

1. **每图都有明确目的** - 知道这张图要说明什么
2. **添加置信区间** - 预测图必须显示误差范围
3. **标注关键数值** - 柱状图在柱顶显示数值
4. **计算相关系数** - 散点图必须显示R²
5. **数据标准化** - 热力图和平行坐标图要标准化
6. **高分辨率保存** - 至少300 DPI

### ❌ 避免的错误

1. ❌ 没有坐标轴标签
2. ❌ 没有图例说明
3. ❌ 使用默认配色
4. ❌ 分辨率太低
5. ❌ 图表不清晰

## 🎯 C题典型应用

### 场景1: 用户行为预测
使用 `折线图_时间序列.py`
- 展示历史数据趋势
- 添加预测曲线和置信区间
- 标注关键时间点

### 场景2: 推荐算法对比
使用 `柱状图_分组对比.py`
- 对比多个算法的准确率、召回率等指标
- 在柱顶显示具体数值
- 使用分组柱状图展示多指标

### 场景3: 特征工程
使用 `热力图_相关矩阵.py`
- 计算所有特征的相关系数
- 找出强相关特征对
- 用于特征选择和降维

### 场景4: 客户细分
使用 `平行坐标图_高维数据.py`
- 展示不同客户群体的多维特征
- 用颜色区分不同类别
- 标准化数据便于对比
