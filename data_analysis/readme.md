## 🎯 我应该用哪个图表？

### 一、数据统计图：揭示规律，支撑论点

**定位**：论文结果的证据基石，客观展示数据中的模式、趋势、差异与关联

| 图表类型 | 核心目的 | O奖应用场景 | O奖制图要点⭐ |
|:--------|:--------|:-----------|:------------|
| **折线图**<br>`折线图_时间序列.py` | **展示时间序列变化趋势**<br>• 变量随时间如何演化？<br>• 预测的未来趋势是什么？ | • **A题**：温度、物理量随时间演化<br>• **C题**：经济指标、疫情预测曲线 | ✅ **必须添加置信区间**<br>✅ 多条线用不同颜色/线型<br>✅ 标注关键点（峰值、交点） |
| **柱状图**<br>`柱状图_分组对比.py` | **比较类别间数值差异**<br>• 哪个方案指标最优？<br>• 因素影响如何排序？ | • **所有题型**：模型性能对比<br>• **E/F题**：政策方案成本收益对比 | ✅ **按数值排序**<br>✅ **柱顶标注具体数值**<br>✅ 分组柱状图对比多指标 |
| **散点图**<br>`散点图_相关性分析.py` | **揭示变量间相关性**<br>• 两因素是否有关联？<br>• 线性还是非线性？ | • **C题**：用户行为特征相关性<br>• **B题**：实验数据与模型拟合 | ✅ **必须显示R²和拟合线**<br>✅ 用颜色区分数据子集<br>✅ 标注异常点 |
| **热力图**<br>`热力图_相关矩阵.py` | **呈现相关系数矩阵**<br>• 哪些变量强相关？<br>• 二维密度如何分布？ | • **C题**：特征相关性矩阵<br>• **E题**：地理指标强度分布 | ✅ **清晰色阶+数值标注**<br>✅ 聚类排序让模式明显<br>✅ 自动找出强相关特征对 |

**脚本位置**：`visualization/` 文件夹  
**快速开始**：`cd visualization && python 折线图_时间序列.py`

| **框架图** | **展示模型组成与交互**<br>• 模型由哪些部分构成？<br>• 各模块如何交互？ | • **A/B题**：耦合系统相互作用<br>• **F题**：社会经济要素反馈 | • 模块化绘制<br>• 不同形状/颜色区分功能<br>• 连接线标注关系性质 |

## 💎 O奖核心思维：从"展示数据"到"讲述洞察"

O奖论文的图表不仅"画得好"，更在于"用得巧"：

### 三大黄金法则

1. **每图一论点** 📌  
   动笔前明确：这张图要为论文的哪个核心论点提供视觉证据

2. **服务于叙事** 📖  
   图表出现顺序遵循论文逻辑叙事线，层层递进导向结论

3. **降低阅读成本** ⚡  
   通过精心的标注、配色和排版，让评委**5秒内**看懂核心信息

### 五个必须做到⭐

| 图表类型 | O奖必备要素 |
|:--------|:----------|
| 折线图 | ✅ 置信区间 + 关键点标注 |
| 柱状图 | ✅ 柱顶数值 + 按大小排序 |
| 散点图 | ✅ R² + 拟合线 + p值 |
| 热力图 | ✅ 数值标注 + 找出强相关对 |
| 平行坐标图 | ✅ 数据标准化 + 颜色分类 |

## 🚀 快速开始

### 1. 安装依赖
```bash
pip install pandas numpy matplotlib seaborn scikit-learn scipy
```

### 2. 运行示例
```bash
# 可视化
cd visualization
python 折线图_时间序列.py

# 数据预处理
cd preprocessing
python 缺失值处理.py

# 工具函数
cd utils
python 数据读取工具.py
```

### 3. 修改数据
打开脚本，找到 `# 生成示例数据` 部分，替换成你的数据即可
