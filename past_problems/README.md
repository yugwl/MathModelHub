## 💡 使用建议

1. **选题参考**：根据历年数据，C/E/F题选择率最高（~70%）
2. **算法学习**：统计发现AHP、ARIMA、熵权法、K-means出现频率最高
3. **论文学习**：重点学习O奖论文的摘要写作、图表设计、逻辑结构
4. **趋势分析**：近年来越来越重视数据分析、机器学习方法
