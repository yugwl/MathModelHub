**为什么选择C题？**
- C题是ICM（跨学科建模竞赛）中最热门的题型
- 选题率约占总参赛队伍的30-35%
- 题目通常涉及数据分析、预测模型和现实问题
- 适合有编程基础和数据分析能力的团队
