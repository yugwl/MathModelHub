## 一、问题分析与建模思路

### 1.1 问题回顾

题目要求：
> 为每个国家开发奖牌计数模型（至少涵盖金牌数与总奖牌数），需包含模型预测的不确定性/精度估计以及模型表现衡量指标。

### 1.2 为什么选择回归模型？

| 分析维度 | 数据特征 | 推荐方法 |
|----------|----------|----------|
| 预测目标 | 连续数值（奖牌数：0, 1, 2, ... 126） | 回归分析 |
| 自变量 | 多个特征（历史成绩、东道主等） | 多元回归 |
| 数据量 | 1435条记录 | 机器学习方法可用 |

**通俗解释**：
- 我们要预测的是一个**数字**（奖牌数），不是"是/否"这样的分类
- 所以用**回归（Regression）**，而不是分类（Classification）
- "回归"可以理解为"根据已知信息推算未知数值"

### 1.3 建模流程图

```
数据加载 → 探索性分析 → 东道主效应验证
                ↓
        回归模型构建
        ├── 线性回归（基线模型）
        ├── Ridge/Lasso（正则化模型）
        ├── 随机森林（集成模型）
        └── 梯度提升（最优模型）
                ↓
        模型评估与比较 → 2028年预测
```

## 三、东道主效应分析

### 3.1 什么是东道主效应？

**定义**：主办国在奥运会上通常会获得比平时更多的奖牌，这种现象称为"东道主效应"（Home Advantage）。

**可能的原因**：
1. 🏠 主场观众支持，运动员士气更高
2. ✈️ 无需长途旅行，身体状态更佳
3. 💰 主办国通常会增加体育投入
4. 🎯 裁判可能存在无意识的倾向性（主观评分项目）
5. 🏟️ 对比赛场地更熟悉

### 3.2 统计验证方法

#### （1）描述性统计

| 分组 | 样本量 | 平均奖牌数 | 中位数 | 标准差 |
|------|--------|-----------|--------|--------|
| 非东道主 | 1405条 | 11.3 枚 | 4 枚 | 18.5 |
| 东道主 | 30条 | 66.9 枚 | 43.5 枚 | 57.2 |

**直观结论**：东道主的平均奖牌数是非东道主的 **6倍**！

#### （2）独立样本t检验

**目的**：检验东道主与非东道主的奖牌数差异是否**统计显著**（即不是偶然的）。

**假设**：
- $H_0$（原假设）：东道主与非东道主的平均奖牌数**无显著差异**
- $H_1$（备择假设）：东道主与非东道主的平均奖牌数**有显著差异**

**t检验公式**：
$$
t = \frac{\bar{X}_1 - \bar{X}_2}{\sqrt{\frac{s_1^2}{n_1} + \frac{s_2^2}{n_2}}}
$$

**符号说明**：
- $\bar{X}_1$, $\bar{X}_2$：两组的样本均值（东道主 vs 非东道主）
- $s_1$, $s_2$：两组的样本标准差
- $n_1$, $n_2$：两组的样本量

**我们的结果**：
- t统计量 = 15.0
- p值 < 0.001（远小于显著性水平0.05）

**结论**：拒绝原假设，东道主效应**统计显著**。

#### （3）效应量（Cohen's d）

**问题**：t检验只告诉我们"差异是否显著"，但**差异有多大**？

**Cohen's d** 用于衡量效应量（effect size）：
$$
d = \frac{\bar{X}_1 - \bar{X}_2}{s_{\text{pooled}}}
$$

其中，合并标准差：
$$
s_{\text{pooled}} = \sqrt{\frac{(n_1-1)s_1^2 + (n_2-1)s_2^2}{n_1+n_2-2}}
$$

**解释标准**：
| Cohen's d | 效应大小 |
|-----------|----------|
| < 0.2 | 微小效应 |
| 0.2 - 0.5 | 小效应 |
| 0.5 - 0.8 | 中等效应 |
| > 0.8 | **大效应** |

**我们的结果**：$d = 2.77$（大效应）

**结论**：东道主效应不仅统计显著，而且**效应量很大**。

### 3.3 东道主效应量化

$$
\text{增幅百分比} = \frac{\bar{X}_{\text{东道主}} - \bar{X}_{\text{非东道主}}}{\bar{X}_{\text{非东道主}}} \times 100\%
$$

计算结果：东道主平均奖牌数比非东道主高 **491.9%**（约5倍）。

## 四、回归模型介绍

### 4.1 线性回归（Linear Regression）

#### 通俗解释

线性回归就是找一条"最佳直线"（高维空间中是"超平面"），使得预测值与真实值的差距最小。

#### 数学公式

$$
\hat{y} = \beta_0 + \beta_1 x_1 + \beta_2 x_2 + \cdots + \beta_n x_n
$$

**符号说明**：
| 符号 | 含义 | 举例 |
|------|------|------|
| $\hat{y}$ | 预测值（预测的奖牌数） | 预测中国2028年获得90枚奖牌 |
| $\beta_0$ | 截距（基准值） | 当所有特征为0时的预测值 |
| $\beta_i$ | 第i个特征的系数 | $\beta_{\text{is\_host}}$ 表示东道主效应的贡献 |
| $x_i$ | 第i个特征的值 | 如 $x_1$ = 上届奖牌数 |
| $n$ | 特征数量 | 我们使用了7个特征 |

#### 系数的解释

| 特征 | 系数（标准化后） | 解释 |
|------|-----------------|------|
| total_rolling3_mean | 13.54 | 近3届平均每增加1个标准差，预测奖牌增加13.5枚 |
| gold_lag1 | 6.21 | 上届金牌数影响显著 |
| is_host | 5.52 | **东道主身份平均增加约5.5枚奖牌**（标准化后） |

### 4.2 正则化回归（Ridge & Lasso）

#### 为什么需要正则化？

当特征之间**高度相关**（如 total_lag1 和 total_rolling3_mean）时，普通线性回归的系数可能**不稳定**。正则化通过添加**惩罚项**来解决这个问题。

#### Ridge回归（L2正则化）

$$
\text{损失函数} = \sum_{i=1}^{m}(y_i - \hat{y}_i)^2 + \alpha \sum_{j=1}^{n}\beta_j^2
$$

**符号说明**：
- 第一项：预测误差（越小越好）
- 第二项：惩罚项（系数的平方和）
- $\alpha$：正则化强度（控制惩罚力度）

**效果**：系数会被"缩小"，但不会变成0。

#### Lasso回归（L1正则化）

$$
\text{损失函数} = \sum_{i=1}^{m}(y_i - \hat{y}_i)^2 + \alpha \sum_{j=1}^{n}|\beta_j|
$$

**与Ridge的区别**：惩罚项用绝对值，而非平方。

**效果**：某些系数会被压缩为0，实现**自动特征选择**。

#### 对比示例

| 特征 | 线性回归系数 | Ridge系数 | Lasso系数 |
|------|-------------|----------|----------|
| total_lag1 | -2.72 | -2.46 | **0.00** |
| total_events | 0.14 | 0.14 | **0.00** |
| is_host | 5.52 | 5.52 | 5.41 |

**观察**：Lasso将 total_lag1 和 total_events 的系数压为0，说明这些特征可能冗余。

### 4.3 随机森林（Random Forest）

#### 通俗解释

想象你有100个"专家"（决策树），每个专家只看数据的一部分，各自独立做出预测。最后取所有专家预测的**平均值**作为最终结果。

这就是"随机森林"——很多棵决策树组成的"森林"。

#### 为什么效果好？

1. **减少过拟合**：单棵树容易过拟合，多棵树平均后更稳定
2. **捕捉非线性**：可以学习复杂的特征交互
3. **鲁棒性强**：对异常值不敏感

#### 特征重要性

随机森林可以告诉我们**哪些特征最重要**：

| 特征 | 重要性 |
|------|--------|
| gold_lag1 | 0.345 |
| total_lag1 | 0.332 |
| total_rolling3_mean | 0.151 |
| is_host | 0.065 |
| total_events | 0.044 |

### 4.4 梯度提升（Gradient Boosting）

#### 通俗解释

如果说随机森林是"100个专家独立投票"，那么梯度提升是"后一个专家专门纠正前一个专家的错误"。

#### 工作原理

```
第1棵树：预测 → 产生误差
        ↓
第2棵树：学习"如何纠正第1棵树的误差" → 新的残差
        ↓
第3棵树：学习"如何纠正前2棵树的残差"
        ↓
...
最终预测 = 所有树的预测之和
```

#### 与随机森林对比

| 对比项 | 随机森林 | 梯度提升 |
|--------|----------|----------|
| 训练方式 | 并行（独立） | 串行（迭代） |
| 目标 | 减少方差 | 减少偏差 |
| 精度 | 好 | **通常更好** |
| 过拟合风险 | 较低 | 较高 |

### 4.5 我们选择的特征

```python
feature_columns = [
    'total_lag1',           # 上一届奖牌数
    'total_lag2',           # 上两届奖牌数
    'gold_lag1',            # 上一届金牌数
    'total_rolling3_mean',  # 过去3届平均
    'is_host',              # 是否为东道主
    'total_events',         # 当届总项目数
    'participation_count',  # 参赛届数
]
```

**为什么选这些特征？**
- 这些是"预测时刻已知"的信息
- 不会造成**数据泄露**（用未来信息预测未来）

## 五、模型评估指标

### 5.1 指标介绍

#### （1）MSE（均方误差）

$$
\text{MSE} = \frac{1}{m}\sum_{i=1}^{m}(y_i - \hat{y}_i)^2
$$

**符号说明**：
- $m$：样本数量
- $y_i$：第i个样本的真实值
- $\hat{y}_i$：第i个样本的预测值

**特点**：对大误差敏感（因为平方）

#### （2）RMSE（均方根误差）

$$
\text{RMSE} = \sqrt{\text{MSE}} = \sqrt{\frac{1}{m}\sum_{i=1}^{m}(y_i - \hat{y}_i)^2}
$$

**特点**：单位与原数据相同（枚），更直观

#### （3）MAE（平均绝对误差）

$$
\text{MAE} = \frac{1}{m}\sum_{i=1}^{m}|y_i - \hat{y}_i|
$$

**特点**：对异常值不敏感

#### （4）R²（决定系数）

$$
R^2 = 1 - \frac{\sum_{i=1}^{m}(y_i - \hat{y}_i)^2}{\sum_{i=1}^{m}(y_i - \bar{y})^2}
$$

**符号说明**：
- $\bar{y}$：真实值的平均值

**解释**：
- $R^2 = 1$：完美预测
- $R^2 = 0$：与简单预测均值一样好
- $R^2 < 0$：比预测均值还差

### 5.2 模型性能对比

| 模型 | RMSE | MAE | R² |
|------|------|-----|-----|
| 线性回归 | 4.58 | 3.00 | 0.945 |
| Ridge | 4.58 | 3.00 | 0.945 |
| **Lasso** | **4.45** | **2.88** | **0.948** |
| 随机森林 | 5.20 | 3.17 | 0.930 |
| 梯度提升 | 4.60 | 2.97 | 0.945 |

**结论**：
- **Lasso表现最佳**（R² = 0.948）
- 所有模型R²都在0.93以上，说明模型解释力很强
- 平均误差约3-5枚奖牌，对于预测来说是可接受的

### 6.2 Bootstrap置信区间

#### 什么是Bootstrap？

**问题**：模型只能给出一个预测值（点估计），但我们还想知道**预测有多不确定**。

**Bootstrap方法**：
1. 从训练数据中**有放回地抽样**，创建多个"伪数据集"
2. 在每个伪数据集上训练模型
3. 用所有模型对同一样本预测
4. 根据预测值的分布计算置信区间

#### 置信区间公式

$$
\text{95\% CI} = [\text{Percentile}_{2.5}, \text{Percentile}_{97.5}]
$$

**解释**：我们有95%的信心认为真实值落在这个区间内。

### 7.2 泊松分布模型

#### 为什么用泊松分布？

**泊松分布**适合描述"在固定时间/空间内，稀有事件发生的次数"。

首次获奖事件的特征：
- 事件相对稀有
- 各国首次获奖相互独立
- 平均发生率相对稳定

#### 泊松分布公式

$$
P(X = k) = \frac{\lambda^k e^{-\lambda}}{k!}
$$

**符号说明**：
- $X$：随机变量（首次获奖国家数）
- $k$：具体的数量（0, 1, 2, ...）
- $\lambda$：平均发生率（我们计算得 $\lambda = 5.8$）
- $e$：自然常数（约2.718）
- $k!$：k的阶乘

#### 预测结果

| 首次获奖国家数 | 概率 | 累积概率 |
|---------------|------|----------|
| 0 | 0.30% | 0.30% |
| 1 | 1.76% | 2.06% |
| 2 | 5.09% | 7.15% |
| 3 | 9.85% | 17.00% |
| 4 | 14.28% | 31.27% |
| 5 | 16.56% | 47.83% |
| 6 | 16.01% | 63.84% |
| 7 | 13.26% | 77.10% |
| 8 | 9.62% | 86.72% |

**结论**：
- **期望值**：5.8个国家（点估计约6个）
- **90%预测区间**：[2, 10]个国家
- **最可能的结果**：5-6个国家首次获奖

### 9.2 关键公式汇总（LaTeX格式）

**线性回归**：
```latex
\hat{y} = \beta_0 + \sum_{i=1}^{n} \beta_i x_i
```

**t检验统计量**：
```latex
t = \frac{\bar{X}_1 - \bar{X}_2}{\sqrt{\frac{s_1^2}{n_1} + \frac{s_2^2}{n_2}}}
```

**Cohen's d效应量**：
```latex
d = \frac{\bar{X}_1 - \bar{X}_2}{s_{\text{pooled}}}
```

**Ridge回归损失函数**：
```latex
L = \sum_{i=1}^{m}(y_i - \hat{y}_i)^2 + \alpha \sum_{j=1}^{n}\beta_j^2
```

**Lasso回归损失函数**：
```latex
L = \sum_{i=1}^{m}(y_i - \hat{y}_i)^2 + \alpha \sum_{j=1}^{n}|\beta_j|
```

**RMSE**：
```latex
\text{RMSE} = \sqrt{\frac{1}{m}\sum_{i=1}^{m}(y_i - \hat{y}_i)^2}
```

**R²决定系数**：
```latex
R^2 = 1 - \frac{\sum_{i=1}^{m}(y_i - \hat{y}_i)^2}{\sum_{i=1}^{m}(y_i - \bar{y})^2}
```

**泊松分布**：
```latex
P(X = k) = \frac{\lambda^k e^{-\lambda}}{k!}
```

**95%置信区间**：
```latex
\text{95\% CI} = [\text{Percentile}_{2.5}, \text{Percentile}_{97.5}]
```
