```
【线性回归】

 x₁,x₂,...    β₀+Σβᵢxᵢ      ŷ

【随机森林】
输入特征 ──→ ┌─ 树1 ─┐     平均     预测值

             └─ 树n ─┘

【梯度提升】
输入特征 ──→ 树1 ──→ 残差 ──→ 树2 ──→ 残差 ──→ 树3 ──→ ... ──→ 累加 ──→ ŷ
                   (纠错)         (纠错)
```

### 2.3 建议检索的具体文献（示例）

以下是一些可能有用的文献方向，请在数据库中搜索类似论文：

#### 东道主效应相关

```
搜索词: "Olympic home advantage" OR "host country medal"
筛选: 2010年以后, 被引次数>20

可能找到的文献类型：
- 分析历届奥运会东道主表现的统计研究
- 东道主效应的心理学/社会学解释
- 主场优势在不同体育项目中的差异
```

#### 奥运预测模型相关

```
搜索词: "predicting Olympic medals" OR "Olympic performance forecasting"

可能找到的文献类型：
- 使用机器学习预测奖牌的研究
- 基于GDP、人口等宏观指标的预测模型
- 时间序列方法在体育预测中的应用
```

#### 机器学习方法相关

```
搜索词: "Gradient Boosting" + "prediction" + "sports"
或: "Random Forest" + "regression" + "forecasting"

可能找到的文献类型：
- 梯度提升/随机森林方法论论文
- 这些方法在体育分析中的应用案例
```

### 2.4 文献引用格式

美赛论文通常使用 **IEEE格式** 或 **APA格式**，示例：

**IEEE格式**：
```
[1] J. Smith, "Predicting Olympic Medal Counts Using Machine Learning," 
    Journal of Sports Analytics, vol. 5, no. 2, pp. 123-145, 2023.
```

**APA格式**：
```
Smith, J. (2023). Predicting Olympic Medal Counts Using Machine Learning. 
    Journal of Sports Analytics, 5(2), 123-145.
```

### 2.5 引用注意事项

1. **数量**：美赛论文建议引用 **8-15篇** 参考文献
2. **时效性**：优先选择 **近5年** 的文献
3. **权威性**：优先选择 **高被引** 或 **顶级期刊** 的论文
4. **相关性**：确保引用的文献与论文内容直接相关
5. **避免**：不要引用百度百科、知乎等非学术来源
